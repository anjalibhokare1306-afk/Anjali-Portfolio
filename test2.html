Projects

Smart Child Immunization Portal

Tech Stack: ASP.NET, C#, .NET Framework, HTML, CSS, JavaScript, SQL Server

Duration: July 2025 - November 2025

Smart Child Immunization Portal is a role-based web application designed to digitize and streamline the child vaccination process. The platform connects parents, hospitals, and administrators on a single system for efficient immunization management and accurate vaccination tracking.

Parents can book vaccination slots and view immunization details, while hospitals securely manage vaccine records and update vaccination statuses. Administrators use a centralized dashboard to monitor system operations and ensure smooth coordination among all stakeholders.

The system includes automated vaccination reminders, secure storage of medical records, and digital certificate generation. These features reduce manual effort, minimize errors, and improve overall immunization workflow efficiency.

Pesticide Usage Tracker

Tech Stack: HTML, CSS, JavaScript, Backend (API), Database

Duration: October 2025 — December2025

Pesticide Usage Tracker is a web-based application developed to monitor and manage pesticide usage in agricultural practices. The system helps farmers and agricultural officers record pesticide application details and track usage patterns to promote safer and more responsible farming.

The platform allows users to log pesticide types, quantities, application dates, and crop information. Stored data can be reviewed over time to analyze trends, prevent overuse, and support data-driven decision-making.

By providing structured tracking and historical insights, the system improves compliance with agricultural guidelines, reduces environmental risks, and supports sustainable pesticide management.

Intelligent Multi-Modal Assessment System for Cleft Lip & Palate

Tech Stack: Python, PyQt5, Flask, SQLite

Duration: January 2026 – Ongoing

This project focuses on developing an intelligent multi-modal assessment system for cleft lip and palate patients using machine learning techniques. The system aims to assist clinicians by providing objective and consistent evaluation through the integration of multiple assessment modalities.

The application analyzes facial images, speech characteristics, and clinical parameters to assess patient conditions more accurately. A standalone desktop interface is built using PyQt5, while Flask is used for backend processing and data handling.

By combining visual, audio, and clinical data, the system improves assessment reliability, reduces subjectivity in manual evaluation, and supports better clinical decision-making for cleft lip and palate treatment planning.
